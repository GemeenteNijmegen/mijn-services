# Meta patroon: archiveren

**Mijn-Services naam:** Dit is geen Mijn-Services, maar een meta patroon

**Links:**
- 

Het archiveren patroon bestaat uit 1 patronen:
- Zaken, Besluiten en Document archiveren
- Zaken, Besluiten en Document vernietigen

Ook van toepassing op andere patronen zoals: contactomenten, ...?


## Zaken, Besluiten en Document archiveren

**Doel:**
- Zaken, Besluiten en Documenten worden volgens de juiste eisen gearchiveerd.

**Hoe:**
- De Zaken API, Besluiten API, Documenten API kunnen archiefwaardig worden ingericht. Deze kunnen dus als archief worden gebruikt.

## Zaken, Besluiten en Document vernietigen

**Doel:**
- Zaken, Besluiten en Documenten worden na de bewaardermijnen vernietigt.

**Hoe:**
- De Zaken API, Besluiten API, Documenten API kunnen archiefwaardig worden ingericht. Deze kunnen dus als archief worden gebruikt.
- op basis van de archieftermijnen kan een rapport worden gemaakt voor vernietiging van zaken, besluiten en document
